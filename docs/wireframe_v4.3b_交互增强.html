核心概念
CPS (Cost Per Sale)
每笔交易的营销成本占比。CPS = 渠道花费 / 交易额。CPS 越低越好。例如 CPS=30% 意味着每100元交易额中30元是营销成本。
M0 / T0 / M1+
T0: 当月完成首次借款的用户。M0: 当月首次获得授信的用户(范围大于T0)。M1+: 前月授信本月才完成首借的用户。M0/T0系数约1.49，表示M0群体约为T0的1.5倍。
MMM / Adstock / Hill饱和
MMM: Marketing Mix Model，通过回归分析量化各渠道对业务指标的边际贡献。
Adstock: 广告的滞后效应。本周投放的广告不仅本周有效，后续几周仍有残余效果。用 Weibull 函数建模衰减曲线。
Hill饱和: 花费越多，单位花费的效果越低(边际递减)。用 Hill 函数建模，K=半饱和点花费，n=曲线陡度。
护栏指标 (FPD30/终损率/LTV)
FPD30: 首次逾期30天率，衡量短期资产质量。
首借终损率: 首次借款的最终坏账率。
复借终损率: 复借用户的最终坏账率。
渠道LTV: 用户生命周期价值。
这些指标从上传数据中自动识别，用于约束预算分配不超过风险阈值。
