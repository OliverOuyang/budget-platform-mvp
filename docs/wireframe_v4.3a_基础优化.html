计算逻辑说明 V4.3: 新增
此页说明预算推算的完整计算流程，方便理解结果含义和参数影响。适合汇报说明和新人上手。
一、总体流程
📋 上传数据→📊 提取历史系数→🎯 设定目标+参数→🔢 计算引擎→📈 结果展示
二、核心公式 (Table 1: 渠道维度)
渠道花费 = 总花费 × 渠道占比
T0交易额(亿) = 渠道花费 ÷ CPS ÷ 10000
T0申完量 = 渠道花费(元) ÷ T0申完成本
当月首登M0 = T0交易额 × M0/T0系数 (6月均值=1.49)
1-3授信量 = 申完量 × 1-3过件率
三、客群汇总 (Table 2)
整体首借 = 初审M0 + 初审M1+ + 非初审户
全业务CPS = (投放花费 + RTA费用) ÷ 整体首借
四、目标方案如何影响计算
目标修改的参数公式对结果的影响
💰 降成本各渠道CPSCPS × (1 - 幅度)CPS↓ → T0交易额↑ → 首借↑
⭐ 提质量各渠道1-3过件率过件率 × (1 + 幅度)过件率↑ → 授信量↑ → 结构优化
📈 提规模总预算预算 × (1 + 幅度)花费↑ → 各渠道等比↑ → 交易↑
五、关键系数来源
系数计算方式当前值稳定性
M0/T0比值最近6个月月度比值的均值1.49CV=1.5%
存量M0 CPS最近3个月月度CPS均值34.4%下降趋势
渠道CPS用户输入/MMM推荐各渠道不同—
过件率用户输入/历史值加权14.6%—
免费渠道特殊处理：CPS=0(无投放成本)，T0交易额使用历史数据外推而非公式计算。MMM中作为organic变量处理(有Adstock无Hill饱和)。
